# Create Custom App

## Установка

```
npm i -g create-custom-app
```

## Начало работы

```
create-custom-app
```

## Кастомные схемы

### Стандартные поля

- **folders**: (Array|String) Набор папок
- **files**: (Array|String) Набор файлов

### Доволнительные возможности

- **prompt**: (Array) Дополнительные вопросы
- **propmtThen**: (Function) Обработчик вопросов
